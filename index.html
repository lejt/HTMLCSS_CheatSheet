HTML & CSS Cheat Sheet

HTML:

Document Loadout

Tag Name Description
<!DOCTYPE html> Document Type Indicating the current .html file is using HTML
<html> HTML Everything enclosed is HTML
<head> Head Head contains links to .css file, other stylesheets like font and title
<link> Link <link href="file location of css" rel="stylesheet">
<title> Title Changes title of page on tab
<body> Body Should contain block of all other HTML code
<!--> Comment HTML one line comment <!-- COMMENT -->

General Body Tags

Tag Name Description
<h1,h2,h3,h4,h5,h6> Headings Headings of each size
<ul> Unordered List Lists such as bullet points
<ol> Ordered List Lists such as numbered points
<li> List Bullet or numbered point in between ul/ol tag
<p> Paragraph Contains paragraphs
<a> Hyperlinks <a href="URL" (optional- target="_blank" for open in new tab)>Word/ImageUsedAsHyperLink</a>

Media tags

Tag Name Description
<img> Image <img src="URL" />
<video> Video <video src="URL" (optional- width="" height="") controls>Video not supported</video>
<audio> Audio <audio><source src="URL" (optional- type="audio/mp3" autoplay)</audio>
<em> Embedded Legacy code for all media types: video, audio, img
<figure> Figure For media with a caption <figure><img (or any media) src="URL"><figcaption>WordsForLabel</figcaption></figure>

Container Tags for Organization

Tag Name Description
<header> Header For introductory content or nav links
<nav> Navigation Container used to section general nav links in header
<div> Division General block container for sectioning
<span> Span Inline container used to markup part of a text
<article> Article Usually a container for different paragraphs
<footer> Footer Container for bottom section of HTML webpage

Specificity Attributes

Tag Name Description
class Class Used to specify a class for element(tag) <tag class="">
id Identity Cannot have more than one element(tag) with same ID <tag id="">

Table tags

Tag Name Description
<table> Table Wrapper element for all HTML tables
<thead> Table Head Set of rows for column headers
<tbody> Table Body Like the body wrapper but for the table main code
<tr> Table Row Table row
<th> Table Head Headings for table
<td> Table Data Used to input data into each table block
<tfoot> Table Foot Set of rows for footer in table

Table Attributes

Tag Name Description
colspan Column Span Defines how many rows a column should span <td colspan="">DataInput</td>
rowspan Row Span Defines how many columns a row should span <td rowspan="">DataInput</td>
scope Scope For Table Head, defines if heading in row or column <th scope="">DataInput</th>

CSS Loadout

Tag Name Description
@import Import At the top, importing fonts, ex. @import url()
