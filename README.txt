# Codeacademy Cheat Sheet Project

The following contains a cheat sheet for general HTML and CSS tags and attributes.

## Table of Contents
* [General Information](#general-information)
* [Programming Tools Used](#programming-tools-used)
* [Features](#features)
* [Usage](#usage)

## General Information
The intention of this project is to practice making a webpage and organizing a layout of all the HTML and CSS learnt throughout Codecademy.

## Programming Tools used
* HTML5
* Git v2.33.1.windows.1
* Visual Studio Code v1.61.1

## Features
### Implemented
* Nav bar to move to each section of cheat sheet
* Multiple html table usage along with identifiable containers for organization
* Footer for additional hyperlink information
### Development Ideas
* Implement side by side tables for more space efficiency
* Include hyperlinks of each tag/attribute linked to MDN or similar 
* Use of more complex css for aesthetic appeal such as transitions

## Usage
To run this program, navigate to directory in terminal and open html file with the following command:
> open index.html
